Token Proposal

Real OS UI HTML uses [data-button-group] + .button-group-item (with .button-group-selected-item for the active option). Items share borders via border-left: 0 on non-first items. Proposal softens the borders (primary → neutral-default on rest, primary only on selected), drops height to 36px, and bumps radius to 8px.
