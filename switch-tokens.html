Token Proposal

Real OS UI uses a native <input type="checkbox" data-switch> styled via :before (track) + :after (thumb) pseudo-elements. Proposal shrinks the track (50×32 → 40×24), drops the border (track is now a plain pill), and reduces thumb to 20px to match modern toggles (iOS, Material, Tailwind UI).
