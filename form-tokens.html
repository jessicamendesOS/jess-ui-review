Token Proposal

Real OS UI uses <form data-form class="form card"> with .columns.columns2 for 2-column layouts, [data-label] + .mandatory for required fields, .input-text / .input-email wrappers around .form-control[data-input]. Proposal softens the card (radius 4→12px, padding 24→32), adds focus halo, and widens row spacing for breathing room.
