Token Proposal

Real OS UI HTML uses the Splide-based .osui-carousel wrapping .osui-carousel__track + .splide__list slides + .splide__pagination dot strip + .splide__arrow prev/next buttons. Proposal softens the track corner (4→12px), swaps the dot scale trick for a bar-style active indicator (width animates from 8 → 28px), and lifts the arrows with a deeper shadow.
