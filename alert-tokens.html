Proposed changes — Alert · based on M3 + Radix

CSS-only. HTML structure unchanged. Uses existing tokens or overridden token defaults.

Property Current Proposed Token Rationale
outer border 1px solid removed token-border-size-025 → 0 M3 banners use no outer stroke — the tonal container background provides sufficient visual containment
border-left-width (accent) 3px 4px token-scale-100 Radix Callout standard: 4px left accent rule — aligns with accessible contrast thresholds for non-text indicators
border-radius 8px 12px token-border-radius-200 → 12px M3 Medium shape = 12dp; also matches Radix --radius-5 (12px) used on Card and Callout
gap (icon ↔ message) 12px 16px token-scale-400 (was scale-300) M3 Banner horizontal spacing = 16dp — standard gap between the leading icon and body content
box-shadow none M3 Level 1 — M3 Elevated Banner variant uses Level 1 elevation to float the alert above page content
icon container none (transparent) 36×36 circle, 16% tonal token-border-radius-full M3 tonal icon container: icon sits in a color-matched pill at ~16% opacity — no extra markup needed
