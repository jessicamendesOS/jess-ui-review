Token Proposal

Real OS UI uses a native <input type="checkbox" data-checkbox> styled via :before (box) + :after (checkmark) pseudo-elements. Proposal shrinks the visible box (22→18px), thickens the border to 2px for better high-DPI rendering, and adds a focus halo for keyboard a11y.
