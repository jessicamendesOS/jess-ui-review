Token Proposal

Real OS UI uses a native <input type="checkbox" data-checkbox> styled via :before (box) + :after (checkmark) pseudo-elements. The proposed box uses the same border, radius, and background tokens as .form-control inputs (1px, 8px radius, token-border-input-default), with the same 24×24px hit / 22×22px visible geometry as New tokens, plus a focus halo for keyboard a11y.
