Token Proposal

Replace 2.28 palette classes with semantic tokens for background and text, and introduce a display font-size token.

Property Component Current (2.28) Proposed Token Rationale
background (primary) .background-primary color-primary (#1068eb) token-semantics-primary-base (#105cef) --token-semantics-primary-base The 2.28 color-primary is a raw palette value. token-semantics-primary-base expresses intent — the primary brand action — and supports theming without touching component classes.
background (success) .background-success color-success (#29823b) token-semantics-success-base (#29823b) --token-semantics-success-base In 2.28 .background-success already points at the --color-success palette token. The 2.29 proposal promotes it to a semantic token so it can diverge from the raw palette in theming without touching component classes.
background (error) .background-error color-error (#dc2020) token-semantics-error-base (#dc2020) --token-semantics-error-base Same hex, different layer. Shared with Badge, Alert, and Toast — one semantic token controls the error red across the entire system instead of each component hard-coding --color-error.
background (warning) .background-warning color-warning (#e9a100) token-semantics-warning-base (#e9a100) --token-semantics-warning-base 2.28 already separates warning from the orange palette via --color-warning. The 2.29 token lifts that into the semantic layer so theming can override warnings without cascading through palette classes.
color (text) .text-neutral-0 color-neutral-0 (#ffffff) token-text-inverse (#ffffff) --token-text-inverse token-text-inverse is semantically correct for text on colored backgrounds and respects dark-mode or high-contrast overrides automatically.
font-size (number) .font-size-display font-size-display · --font-size-display (36px) token-font-size-900 (2.25rem · 36px) --token-font-size-900 2.28's .font-size-display resolves to a single --font-size-display variable. 2.29 promotes it into the ordinal type scale as token-font-size-900, paired with font-weight-regular and line-height-1200, so display typography shares the same scale tokens as the rest of the system.
