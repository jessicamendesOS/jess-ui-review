Token Proposal

Real OS UI drives visuals via inline CSS custom properties on .osui-progress-circle — no modifier classes for semantic variants. Internal structure is an osui-inline-svg wrapper with two <circle> elements (__trail-path + __progress-path) plus an absolutely-positioned __content overlay carrying a .font-size-display utility span. Rounded line-caps are hardcoded in both 2.28 and 2.29 via .progress-circle svg { stroke-linecap: round }.
