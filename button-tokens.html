Token Proposal

Real OS UI HTML uses .btn + variant modifiers (.btn-primary, .btn-success, .btn-error, .btn-cancel) and size modifiers (.btn-small, .btn-large). Proposal shrinks default height (40→36px), softens the radius (4→8px), drops font-weight a notch (600→500), and replaces the filter: brightness hover with a cleaner shadow + ring.
