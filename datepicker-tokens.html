Token Proposal

OSUI DatePicker wraps Flatpickr as .osui-datepicker (input) + .flatpickr-calendar.osui-datepicker-calendar (popup). Calendar structure: month header grid (date | prev | next), weekday strip, 7-col day grid, optional time picker, optional today button. Proposals add elevation via shadow token, soften radii, neutralise the month/year text, modernise day cells to squircle shape, tokenise the today cell border, and convert the today button and AM/PM button to consistent outline pills.
